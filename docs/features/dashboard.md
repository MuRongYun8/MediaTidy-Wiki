# 仪表盘

仪表盘是 MediaTidy 的首页，提供系统状态、任务统计和快捷操作的全局概览。

![MediaTidy 仪表盘与应用缓存管理](/media-tidy-dashboard.png)

## 概览面板

### 最近任务

显示最近 8 条任务记录（包括排队中、运行中、已完成、失败的任务），每条任务显示：

- 任务类型（整理 / STRM / 清理空目录）
- 当前状态和进度
- 开始时间和耗时

### 定时任务

列出所有已注册的定时任务，显示：

| 字段 | 说明 |
|------|------|
| `name` | 任务名称 |
| `type` | 任务类型（organize / strm / clean_dirs） |
| `cronExpr` | Cron 表达式 |
| `enabled` | 是否启用 |
| `nextRun` | 下次执行时间 |

### Emby 服务器状态

显示已配置的 Emby 服务器的在线状态：

- 服务器名称和地址
- 在线/离线状态
- Emby 版本号
- 电影数量和剧集数量

### 115 网盘账号

显示首个 115 账号的用户信息：

- 用户名和头像
- VIP 等级
- 总空间、已用空间、剩余空间

## 活动统计

### 整理统计

| 指标 | 说明 |
|------|------|
| **今日/累计任务数** | 整理任务执行次数 |
| **今日/累计文件数** | 成功处理的文件数 |
| **电影 / 剧集** | 按媒体类型的文件分布 |
| **洗版升级** | 通过质量对比升级的文件数 |
| **错误数** | 处理失败的文件数 |

### STRM 统计

| 指标 | 说明 |
|------|------|
| **今日/累计任务数** | STRM 生成任务执行次数 |
| **今日/累计文件数** | 总处理文件数 |
| **已生成** | 成功生成的 .strm 文件数 |
| **已跳过** | 跳过的文件数（已存在等） |
| **错误数** | 生成失败的文件数 |

### 每日趋势

以图表形式展示近期每天的整理和 STRM 任务文件数，直观反映工作量变化。

### 磁盘用量

异步统计整理目标目录和 STRM 输出目录的磁盘使用情况：

- 各目录的文件总大小和文件数量
- 支持手动刷新（后台异步执行，不阻塞页面）

## 任务队列

所有任务通过全局队列统一调度管理，确保同类型任务顺序执行。

### 队列操作

- **查看队列**：显示排队中、运行中和已完成的所有任务
- **取消任务**：取消指定的排队中或运行中的任务
- **清空等待**：一键取消所有排队等待中的任务（不影响正在运行的任务）
- **实时进度**：通过 SSE（Server-Sent Events）实时推送任务状态变化

### 定时任务管理

| 操作 | 说明 |
|------|------|
| **查看** | 列出所有注册的定时任务及其 Cron 表达式、启用状态、下次执行时间 |
| **启用/禁用** | 切换单个定时任务的启用状态 |
| **修改频率** | 更新任务的 Cron 表达式 |
| **手动触发** | 立即执行一次定时任务（不影响下次计划时间） |
| **删除** | 移除指定的定时任务 |

## 工具箱

MediaTidy 内置实用调试工具，帮助验证配置效果。

### 解析测试

输入文件名，查看完整的识别流程：

- **文件名解析**：展示 guessit 解析结果（标题、年份、季/集号、分辨率、编码等）
- **渲染词替换**：展示自定义渲染词的替换效果
- **TMDB 匹配**：展示 TMDB 搜索和匹配结果
- **分类结果**：展示分类引擎的匹配结果
- **命名预览**：展示最终的文件名和文件夹名
- **支持临时规则**：可以在不修改正式配置的情况下测试临时的渲染词、季集映射和发布组规则

### 正则验证

输入正则表达式和测试字符串，验证匹配结果。用于调试自定义渲染词等正则规则。

### 评分对比

输入两个文件名，对比它们的质量评分：

- 按选定的质量评分规则计算两个文件的各维度得分
- 先展示合格/不合格状态；一方不合格时，合格资源直接胜出，不按总分决定
- 展示全部拒绝原因，例如轨道数量超限或没有匹配的中文轨道
- 展示分辨率、编码、HDR、音频、文件大小、容器总码率和视频流码率等各项得分
- 展示命中的音轨或字幕轨、轨道总数、`priority` 等级以及与理想轨道数量的距离
- 开启 ffprobe 提取后，需要选择可访问的实际文件；只输入文件名时只能使用文件名中可识别的信息

评分对比工具不会扫描文件所在目录，因此不统计同目录外挂字幕。需要验证外挂字幕条件时，以实际整理结果为准。
