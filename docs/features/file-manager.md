# 文件管理

MediaTidy 内置 Web 文件浏览器，提供统一的虚拟路径视图来管理所有云盘和挂载点。

## 统一虚拟路径

所有存储源在文件管理器中以统一的虚拟路径呈现：

```
/                           # 虚拟根目录
├── 我的115/                # 115 账号 A
│   ├── 电影/
│   └── 电视剧/
├── 家庭115/                # 115 账号 B
│   └── 动漫/
└── 本地存储/               # 通用挂载点（如 CD2 挂载的本地目录）
    └── strm/
```

每个 115 账号和通用挂载点都作为顶级目录显示，路径格式为 `/<显示名称>/路径...`。

![文件管理器目录与文件列表示例](/screenshots/file-manager-overview.png)

*顶部目录对应已配置的账号或挂载点。进入目录后可以搜索、勾选文件，并使用右上角按钮新建目录或刷新列表。*

### 路径解析

系统根据虚拟路径的第一级目录自动路由到对应的存储后端：

| 路径类型 | 识别方式 | API 通道 |
|----------|----------|----------|
| **115 账号** | 匹配已配置的 115 账号名称 | 优先 115 原生 API，CD2 备选 |
| **通用挂载** | 匹配已配置的通用挂载名称 | 仅通过 CD2 |
| **根目录** | `/` | 列出所有账号和挂载点 |

## 文件操作

支持以下操作（通过 CD2 或 115 API 执行）：

| 操作 | 说明 |
|------|------|
| **浏览** | 递归浏览目录结构 |
| **搜索** | 按关键词搜索指定目录下的文件 |
| **新建文件夹** | 在当前目录下创建子文件夹 |
| **重命名** | 重命名文件或文件夹 |
| **移动** | 移动文件到其他目录 |
| **复制** | 复制文件到其他目录 |
| **删除** | 删除文件或文件夹 |
| **下载链接** | 获取文件的直链下载 URL |

## 手动整理

在文件管理器中选择文件后，可直接触发手动整理：

1. 勾选要整理的文件/目录
2. 点击「手动整理」按钮
3. 选择一个已有的整理方案
4. 指定 TMDB ID 和媒体类型（movie / tv）
5. 系统按方案配置的命名规则执行转移

手动整理的结果同样记录在[整理记录](/features/organize#整理记录)中，支持后续还原和重新识别。

## 通用挂载点

除了 115 账号，还支持添加「通用挂载点」来管理非 115 的存储：

- CD2 挂载的本地目录
- CD2 挂载的其他网盘（阿里云盘、夸克等）
- 任何在 CD2 中可访问的存储

在「网盘配置」中添加通用挂载点，指定 CD2 挂载路径和显示名称即可。

::: warning 注意
通用挂载点**仅通过 CD2** 进行文件操作，不支持 115 原生 API。因此部分仅 115 API 支持的功能（如离线下载）对通用挂载不可用。
:::
